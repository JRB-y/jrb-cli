## How to use ?
------------------
### Install a project
`$ jrb-cli new <type> <folder>`

#### type:
Type: `string`\
required: `true`

Type of the project can be `vue` , `express` or `express-vue`.

#### folder:
Type: `string`\
required: `true`

Name of the folder and the project
